What is Booglemarks

This is a SIMBL plugin for Safari/WebKit. Access your Google Bookmarks from main menu natively.

What it Does

  • Autoload All Bookmarks from signed in Google Account
  • Add or Edit page in Google Bookmarks
  • Open Tagged Bookmarks in Tabs

How to Install

  • Install SIMBL
  • Copy unzipped Booglemarks.bundle to ~/Library/Application Support/SIMBL/Plugins/
  • Quit and open Safari
  • Find the Booglemarks on Safari's main menu
